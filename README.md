Django 1.8+ 学习笔记
===

Django 1.8 正式版发布了，这是一个 LTS 长期支持的版本。作为一名 python 菜鸟级业余爱好者，在这里记录学习这个框架的过程和体会。一来是鞭策自己这个懒人，二来通过分享学习过程，希望能够为其他英文不好的 Django 新手提供一点点的帮助。

主要的学习资料：

- [官方的文档](https://docs.djangoproject.com/en/1.8/)，文章编排也将大致按官方文档的编排顺序。
